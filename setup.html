Karaplay Setup

Follow these steps on your phone to get Karaplay working in your car.

1 Open Google Console

Click below to open the YouTube API page in a new tab. Log in with your Google account.

2 Create Your Key

Once the API is enabled, go to Credentials, click + Create Credentials, and select API Key. Copy the long code (starts with AIza...).

3 Send to Car

Paste your code below and press the button. This will add the setup link to your Chrome history.

4 Final Step (In Car)

Now, go to your Car's Infotainment screen. Open Chrome and check your History. Click the entry that looks like "Karaplay - API Key Setup...". The car will instantly save your key!
